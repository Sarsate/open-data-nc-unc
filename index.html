Project Introduction
Open-Data NC is a public website that facilitates acquiring public data from government entities in North Carolina. The website allows people to request, search for, and contribute data, creating a central hub for accessing public records, ultimately leading to a more transparent government. We want to add a kickstarter-like feature to the website through crowdfunding and crowdsourcing, providing addtional incentives for contributing data.
